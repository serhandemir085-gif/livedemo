# LivePlay Demo Stack

Bu klasor, GitHub'a direkt yuklenebilecek statik tanitim sitesidir.

## Icerik

- `index.html`: ana landing page
- `styles.css`: tum koyu tema tasarimi
- `script.js`: scroll reveal efektleri
- `assets/`: gercek oyun goruntuleri

## Kullanım

1. Bu klasoru bir GitHub reposuna yukle.
2. Lokal acmak icin `index.html` dosyasini tarayicida ac.
3. Istersen GitHub Pages veya Netlify/Vercel benzeri static hosting'e koy.

## Not

Sayfadaki gorseller, bu klasordeki zip dosyalarindan cikarilan oyunlardan uretilmistir.
